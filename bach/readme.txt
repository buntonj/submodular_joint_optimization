Documentation of Submodular-Multi Package, v1.0, April 2016
Francis Bach, francis.bach@inria.fr

DESCRIPTION

The submodular-multi package is a Matlab program that implements submodular optimization problems to reproduce the figures of the following paper

F. Bach. Submodular Functions: from Discrete to Continous Domains. Technical report, HAL-01222319-v2, 2015. 


INSTALLATION

The script to reproduce figures is the following:

demo_submodular_multi.m
